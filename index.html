Sistema de Gerenciamento de Tarefas (Python + SQLite)

Projeto em Python que permite gerenciar tarefas com controle de usuários e níveis de acesso. Usuários comuns podem criar e acompanhar suas tarefas, enquanto administradores têm acesso total — podendo visualizar, editar e criar tarefas para qualquer usuário. O sistema funciona via linha de comando (CLI) e utiliza SQLite para persistência de dados.

Principais recursos:
- Login com níveis de permissão (usuário e admin)
- CRUD completo de tarefas
- Registro de ações (logs)
- Banco de dados SQLite integrado
- Código modular e simples, com foco em clareza e aprendizado

Calculadora.java

Este projeto foi criado por mim para treinar programação em Java. Trata-se de uma calculadora de console que realiza as quatro operações básicas: adição, subtração, multiplicação e divisão. O programa exibe um painel em texto mostrando os números e a operação escolhida, formata os resultados com duas casas decimais e inclui validações, como prevenção de divisão por zero. É possível encerrar a calculadora a qualquer momento digitando “X”. Com este projeto, pratiquei lógica de programação, estruturas de repetição, condicionais e organização de código, além de aprender a criar uma aplicação funcional e de fácil uso no console.
